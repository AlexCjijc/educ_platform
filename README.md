Данный проект представляет собой веб‑платформу для онлайн‑обучения, вдохновлённую функциональностью Stepik. Он позволяет создавать курсы, управлять содержимым (главы, подглавы, уроки, тесты и задания), назначать роли преподавателям и редакторам, проводить автоматическую и ручную проверку, общаться в чатах и многое другое.

Проект написан на Python с использованием Flask и SQLAlchemy. На данный момент это MVP (минимально жизнеспособный продукт), который уже включает большинство ключевых возможностей.
📌 Основные функции

    Пользователи

        Регистрация / авторизация (с аватарами)

        Личный профиль с рейтингом

        Настройка профиля (смена пароля, email, аватар)

        Роли на сайте: администратор, модератор, обычный пользователь

    Курсы

        Создание курса с описанием, обложкой, категорией

        Настройка видимости (публичный / приватный по ссылке)

        Ограничение числа студентов и сроков доступа

        Возможность задать лимит времени на прохождение (срок после зачисления)

    Структура курса

        Главы (Chapters) и подглавы (Sections)

        Уроки (Lesson) с поддержкой HTML‑контента и встраивания видео (YouTube, Rutube)

        Тесты (Test) с различными типами вопросов:

            Одиночный выбор

            Множественный выбор

            Упорядочивание

            Соответствие (Match)

            Текстовый ответ (требует ручной проверки)

        Задания (Assignment) с текстовым ответом

    Оценивание

        Автоматическая проверка тестов (кроме текстовых вопросов)

        Ручное оценивание с уведомлениями преподавателям

        Гибкая политика оценивания (масштабирование баллов в зависимости от попытки, принудительная ручная проверка после N попыток)

        Прогресс прохождения уроков

    Приглашения и роли

        Генерация пригласительных ссылок с выбором роли (студент, учитель, редактор)

        Ограничение срока действия и количества использований

        Автоматическое зачисление студентов при принятии приглашения

    Чаты

        Общий чат курса (доступен всем участникам)

        Чат для персонала (преподаватели + редакторы)

        Упоминания пользователей с уведомлениями

    Уведомления

        Опубликование оценки

        Запрос на проверку (для преподавателей)

        Упоминания в чатах

    Администрирование

        Панель администратора (список пользователей и курсов)

        Управление курсами (редактирование, удаление)

🛠️ Технологический стек

    Backend: Python 3.10+, Flask 2.x, Flask‑Login, Flask‑Migrate, WTForms

    База данных: SQLite (по умолчанию), поддерживается переключение на MySQL / PostgreSQL

    ORM: SQLAlchemy

    Шаблоны: Jinja2

    Фронтенд: стандартные HTML/CSS/JS (без отдельного SPA‑фреймворка)

🚀 Установка и запуск
1. Клонирование репозитория
bash

git clone https://github.com/yourusername/stepikish-platform.git
cd stepikish-platform

2. Создание виртуального окружения
bash

python -m venv venv
source venv/bin/activate      # Linux / macOS
venv\Scripts\activate         # Windows

3. Установка зависимостей
bash

pip install -r requirements.txt

    Если файла requirements.txt нет, создайте его на основе импортов: flask, flask-sqlalchemy, flask-migrate, flask-login, wtforms, werkzeug, sqlalchemy и т.д.

4. Настройка переменных окружения

Создайте файл .env или задайте переменные вручную:
bash

export SECRET_KEY="ваш-секретный-ключ"
export DATABASE_URL="sqlite:///stepikish.db"   # опционально

5. Инициализация базы данных
bash

flask db init
flask db migrate -m "initial migration"
flask db upgrade

Либо просто создайте таблицы через консоль Python (если миграции не настроены):
python

from app import app, db
with app.app_context():
    db.create_all()

6. Запуск приложения
bash

python app.py

Приложение будет доступно по адресу http://127.0.0.1:5000.
